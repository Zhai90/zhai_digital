Here are some HTML code for me to keep in mind!

-"em" emphasizes the text, giving it a italic text, but not always!

-"strong" makes the text stand out, by making it bold, but then again, not always!

-"br" makes a line-break tag, without needing to start a new line or using margins/paddings. -"p" is a simple "paragraph" tag.

-"code" makes a Markdown code block, like this

-"ul" is a list tag and it means "unordered-list" meaning there is no exact order.

-"ol" is also a list tag and it means "ordered-list" which instead of bullet points, it has numbers indicating steps.

-"li" are "list-items", which surround the things you want in the list.



Here are some CSS code for me to keep in mind!

-A list of CSS tags and its usages in case I need them: SkillCrush List Google Docs Copy

-CSS uses HTML tags for you to decide how they look, that is called CSS Selectors

-The text after the CSS Selectors and between the brackets ("{}") are called CSS declarations, which consists of a property, and value, and determine how they should look. They also split in 2 parts, a name and a value.

-You can make scrolling "smooth" by using the "scroll-behavior" property, and setting it to "smooth"!

-A few of the most basic CSS properties, are "color", "font-size" and "font-family".

-Some other few useful CSS properties, are "text-align", "font-style", "font-weight", "text-transform", "font-variant", "text-decoration", "line-height", "text-indent", "word-spacing" and "letter-spacing".

-You can easily make animations by using "transition" propierty and "Transform" properties (applicable to Z, X and Y properties).

-In order to apply different styles according to the device criteria, for example, if it's in portrait mode, or on phone, etc, that is where media queries come in.



Visual Design, and color theory!

-There are 4 main color schemes, which are; Analogous, Monochromatic, Triad, and Complementary!



Typography!

Some Typeface moods are; Formal VS Informal, Light VS Dramatic, Modern vs Traditional, and Warm VS Cool.

-Some very popular and used fonts are: Serif Typeface, which is mostly used on headlines and bigger more important text.

Sans-Serif Typeface, which is basically "Serif Typeface", but without the feet, making it look more simple, clean and ez-2-read Usually used for a variety of text types.

Script Typeface, handwriting-like, with a lot of loops and swirls, very fancy. Mostly used for headlines, logos and small details.

Display Typefaces (which this website is using!), can be customized and create almost every single type of mood. They are mostly used on logos, small pieces of text and on headlines.
